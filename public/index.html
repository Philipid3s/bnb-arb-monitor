How this prototype works

This prototype is a fast signal checker for potential BNB Chain arbitrage, not an auto-trading bot.

  1. Loads a watchlist of BNB Chain tokens (for example: CAKE, WBNB, ETH, ADA, DOGE).
  2. Fetches reference spot prices from CoinGecko.
  3. Requests quote-based swap prices from OpenOcean using the same quote token and size.
  4. Computes spread and net spread after configured trade costs.
  5. Flags rows as Opportunity when spread exceeds cost + minimum profit threshold.

Use it to prioritize candidates for deeper execution checks (slippage, gas, latency, and routing quality).
