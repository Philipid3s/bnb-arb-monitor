# Arbitrage Backend

Ce dossier contient la partie backend du projet d'arbitrage sur la BNB Chain.

## Prérequis

- [Node.js](https://nodejs.org/) (version 16 ou supérieure recommandée)
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

## Installation

```bash
npm install
```
ou
```bash
yarn install
```

## Configuration

Créez un fichier `.env` à la racine du dossier `Backend` pour y placer vos variables d'environnement (clés API, endpoints, etc).

## Lancement du backend

```bash
npm start
```
ou
```bash
yarn start
```

## Structure du dossier

- `tokens.js` : Liste des tokens surveillés sur la BNB Chain.
- `src/` : Code source principal du backend.
